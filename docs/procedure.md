# AI補間小説生成 — 作業手順

> 初版：2026-02-27（valentine_chocolate の実験を基に作成）

---

## 概要

`templates/novel_template.md` のフォーマットを使ってドラフトを書き、
Claudeに「補間生成を行ってください。」と一言指示するだけで
約5000字の短編小説を生成するワークフロー。

---

## ファイル命名規則

| ファイル | パス |
|---|---|
| ドラフト | `stories/[タイトル]_draft.md` |
| 完成版 | `stories/[タイトル].md` |

---

## 手順

### Step 1 — ドラフトを作る

`templates/novel_template.md` を参考に、`stories/[タイトル]_draft.md` を新規作成する。

埋めるべきブロックは以下の通り：

| ブロック | 内容 | 目安 |
|---|---|---|
| `[SCENE_n]` | 次シーンの場面概要（口語メモでOK） | 1〜3行 |
| `[BLOCK_n]` | 物語の核となる文章 | 300〜500字 |
| `[REF_n]` | 補間に使いたいフレーバー素材（箇条書き） | 自由 |
| `[SENSUALITY]` | 官能パートの想定文字数（該当なければ省略） | 数字のみ |
| `[FINAL_BLOCK]` | 結末の文章（最重要、原文保持される） | 500字推奨 |

**ポイント：**
- BLOCK はざっくりした文でも問題ない。骨格さえあればよい
- FINAL_BLOCK は完成度高めに書く。ここだけは手を抜かない
- ドラフト自体はコミットしておくと後から実験の比較ができる

### Step 2 — Claudeに指示する

ドラフトが完成したら、Claude Code のチャットで以下を送る：

```
補間生成を行ってください。
```

それだけでよい。Claudeはドラフトを読み込み：

1. BLOCK / FINAL_BLOCK を**原文のまま**維持する
2. REFの素材を補間テキストに自然に織り込む
3. SCENE を手がかりにシーン間をつなぐ
4. 完成版を `stories/[タイトル].md` として保存する
5. Commit & Push まで行う

### Step 3 — 確認・調整

生成された完成版を読んで、必要に応じて以下を調整する：

- ドラフトの BLOCK を書き直してやり直す（最も効果大）
- REF に素材を追加してニュアンスを調整する
- FINAL_BLOCK の文章を修正して結末の質を上げる

調整後は再度「補間生成を行ってください。」で再生成する。

## 今日の実験ログ（2026-02-27）

**作品：** `valentine_chocolate`

| 確認項目 | 結果 |
|---|---|
| BLOCKの原文保持 | ✓ 完全保持 |
| FINAL_BLOCKの原文保持 | ✓ 完全保持 |
| REFの補間活用 | ✓ 自然に織り込まれた |
| 文字数 | ✓ 約5000字（テンプレート想定通り） |
| 指示文 | 「補間生成を行ってください。」のみ |

**気づき：**
- BLOCK を 2〜3 箇所に分散させると、物語のリズムが生まれやすい
- REF に「小道具」を入れると、補間テキストの密度が上がる
- SCENE の記述が具体的なほど補間のブレが小さくなる
